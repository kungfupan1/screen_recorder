录屏王软件使用说明书


一、软件概述

1.1 软件类型

录屏王是一款运行于 Windows 操作系统的多媒体屏幕录制工具软件，属于应用软件中的视频采集与录制类软件。本软件基于 Python 语言开发，采用 PySide6 图形框架构建现代化深色主题用户界面，利用 mss 屏幕捕获库进行高性能帧采集，通过 FFmpeg 进行视频编码输出，最终生成 MP4 格式的视频文件。

1.2 开发目的

录屏王的开发旨在为 Windows 用户提供一款轻量、美观、易用的屏幕录制工具。当前市场上的屏幕录制软件普遍存在界面复杂、广告繁多、体积臃肿等问题。录屏王以极简设计理念为核心，提供全屏录制、区域录制、多显示器同时录制、麦克风录音、系统声音录制等核心功能，同时配备会员订阅制商业模式，免费用户可体验基础功能，付费用户可解锁全部高级功能。

本软件的目标用户包括：需要录制教学视频的教育工作者、需要录制操作演示的办公人员、需要录制游戏画面的娱乐用户，以及各类需要将屏幕内容保存为视频文件的 Windows 用户。

1.3 主要功能

录屏王提供以下核心功能：

（1）全屏录制
支持对指定显示器进行完整屏幕画面的录制。当计算机连接多个显示器时，用户可选择其中一个或多个显示器同时进行录制，每个显示器生成独立的视频文件。录制过程采用帧精确同步技术，确保视频帧率稳定。

（2）区域录制
用户可在屏幕上通过鼠标拖拽选取任意矩形区域，软件仅录制该选定区域内的画面内容。此功能为会员专属功能，免费用户需开通会员后方可使用。

（3）多显示器同时录制
当计算机连接两个及以上显示器时，用户可勾选多个显示器同时进行录制，每个显示器独立生成一个视频文件。录制期间显示器选择区域会显示当前检测到的所有显示器及其分辨率信息。

（4）音频录制
提供麦克风录音和系统声音录制两种音频采集方式，用户可独立开关。麦克风用于采集外部语音输入，系统声音用于采集计算机内部播放的音频。两种音源可同时开启，录制完成后自动与视频画面合并。

（5）纯录音模式
当用户取消勾选所有显示器且开启麦克风或系统声音时，软件自动进入纯录音模式，仅采集音频数据并输出音频文件。

（6）暂停与继续
录制过程中用户可随时暂停录制，暂停期间画面停止采集但已录制内容完整保留。恢复录制后视频时间轴无缝衔接。

（7）视频管理
录制完成的视频文件以缩略图卡片形式展示在主界面下方预览区，支持水平滚动浏览。用户可点击卡片播放视频，右键点击可弹出操作菜单，提供播放、另存为、打开所在文件夹、删除等操作。

（8）全局快捷键
提供三组全局快捷键，在任何应用程序获得焦点时均可触发录制控制：Ctrl+F9 开始/停止录制、Ctrl+F10 暂停/继续录制、Ctrl+Esc 退出程序。

（9）会员订阅
软件提供月度、季度、年度、终身四种会员方案，用户通过微信扫码支付完成订阅。会员可享受区域录制、无水印、无限时长等高级功能。同时支持兑换码激活方式。

（10）窗口等比缩放
主界面支持用户拖拽窗口边缘进行等比缩放，界面内所有文字、按钮、图标等元素随窗口大小同步调整，保持布局比例一致。


二、软件使用说明

2.1 运行环境

操作系统：Windows 10 或 Windows 11。
运行方式：双击"录屏王.exe"即可启动（便携版）；或运行官方安装版按向导完成安装后启动。
如使用源码运行，需安装 Python 3.8 及以上版本，并执行 pip install -r requirements.txt 安装依赖。

2.2 主界面介绍

启动录屏王后，主界面从上至下分为以下区域：

（1）标题栏
位于窗口最顶部，显示软件名称"录屏王"，右侧提供最小化按钮和关闭按钮。未激活会员时，标题栏还会显示"免费版"标识。标题栏支持鼠标拖拽移动窗口位置。

【此处插入主界面截图】

（2）状态与时间区
标题栏下方为状态指示区域，包含一个彩色状态点、状态文字（"就绪"/"录制中"/"已暂停"）和录制时长计时器。录制时长以"时:分:秒"格式实时显示（如 00:05:32）。免费用户在计时器右侧会显示"免费限时90S"提示文字。

（3）录制信息区
一块半透明信息面板，显示当前操作的提示文字，如"已选择 1 个显示器，点击开始录制"、"正在录制 2 个显示器..."、"录制完成，共 1 个视频"等。纯录音模式下，此区域会显示音频波形动画。

（4）录制模式选择
提供"全屏录制"和"区域录制"两个模式按钮。默认选中全屏录制。免费用户的区域录制按钮显示为灰色锁定状态并带有 VIP 标识，点击后弹出付费功能提示。

（5）显示器选择
当计算机连接多个显示器时，此区域显示所有检测到的显示器列表，每个显示器以复选框形式展示，标注序号和分辨率（如"显示器 1 (1920x1080)"）。用户可勾选一个或多个显示器同时录制。仅连接一个显示器时此区域自动隐藏。

（6）音频控制
提供"麦克风"和"系统声音"两个开关按钮，分别控制是否采集麦克风输入和系统内部音频。两个开关可独立操作，默认均为开启状态。录制过程中也可实时切换开关状态，立即生效。

（7）控制按钮
三个操作按钮横排排列："暂停"按钮、"录制"按钮（圆形大按钮，中心图标随状态切换）、"停止"按钮。录制前仅录制按钮可用，录制中三个按钮均可用。

（8）已录制文件预览区
底部区域显示已录制的视频缩略图卡片，水平排列，支持滚动浏览。右侧提供文件夹图标按钮，点击可打开视频保存目录。每张卡片显示视频画面缩略图，鼠标悬停时高亮边框，左键点击使用系统默认播放器播放，右键弹出操作菜单。

（9）快捷键提示
界面最底部显示三组快捷键说明文字，方便用户随时参考。

2.3 全屏录制操作步骤

第一步：启动录屏王，确认录制模式为"全屏录制"（默认即为全屏录制）。
第二步：在"选择显示器"区域勾选需要录制的显示器（单显示器用户跳过此步）。
第三步：根据需要开启或关闭麦克风和系统声音开关。
第四步：点击界面中央的圆形录制按钮（或按下 Ctrl+F9）。
第五步：状态文字切换为"录制中"，状态点变为红色，计时器开始计时。
第六步：录制完成后，点击停止按钮（或再次按下 Ctrl+F9）。
第七步：如有音频录制，界面提示"正在合并音视频，请稍候..."，合并完成后自动更新提示。
第八步：录制的视频文件出现在底部预览区，同时保存至用户视频目录下的"录屏王"文件夹。

2.4 多显示器同时录制

第一步：确保录制模式为"全屏录制"。
第二步：在显示器选择区域勾选需要录制的多个显示器。
第三步：信息提示区显示"已选择 N 个显示器，点击开始录制"。
第四步：点击录制按钮开始录制。
第五步：每个选中的显示器各生成一个独立的视频文件。
第六步：停止录制后，所有视频文件同时完成并显示在预览区。

2.5 区域录制操作步骤（会员功能）

第一步：点击"区域录制"按钮切换到区域录制模式。
第二步：显示器选择区域自动禁用（区域录制不支持多显示器选择）。
第三步：点击录制按钮。
第四步：屏幕画面冻结并覆盖一层半透明黑色遮罩，画面中央显示提示文字"拖拽选择录制区域 | 按 ESC 取消"，鼠标光标变为十字形。

【此处插入区域选择界面截图】

第五步：按住鼠标左键拖拽选取需要录制的矩形区域。拖拽过程中，选区内显示原始桌面画面（挖空效果），选区边框为青色高亮，边框上方实时显示选区尺寸和物理坐标信息。
第六步：松开鼠标后，覆盖层消失，录制立即开始。若选区过小（宽或高小于10像素），选择将自动取消，需重新选取。
第七步：如需取消选择，直接按下 Esc 键即可退出区域选择并返回主界面。

2.6 纯录音模式

第一步：确保录制模式为"全屏录制"。
第二步：取消勾选所有显示器。
第三步：确保麦克风或系统声音至少开启一个。
第四步：全屏录制按钮文字自动切换为"仅录音"。
第五步：点击录制按钮开始录音。
第六步：信息提示区显示音频波形动画。
第七步：停止录音后，音频文件自动保存。

2.7 暂停与继续

录制过程中，点击"暂停"按钮或按下 Ctrl+F10 可暂停录制。暂停后状态文字变为"已暂停"，状态点变为黄色，计时器停止。再次点击"继续"按钮或按下 Ctrl+F10 可恢复录制，时间轴无缝衔接。

2.8 视频文件管理

播放视频：左键单击视频缩略图卡片，使用系统默认视频播放器打开播放。

右键菜单操作：右键点击视频缩略图卡片，弹出深色主题操作菜单，提供以下功能：
- 播放：使用系统默认播放器播放该视频
- 另存为：弹出文件保存对话框，可将视频另存到指定位置
- 打开所在文件夹：在系统资源管理器中打开视频文件所在目录
- 删除：弹出确认对话框，确认后删除视频文件并移除预览卡片

【此处插入右键菜单截图】

打开视频目录：点击预览区右侧的文件夹图标按钮，可直接打开视频保存目录（默认路径为当前用户"视频"文件夹下的"录屏王"子目录）。

滚动浏览：当录制的视频较多时，预览区支持鼠标滚轮水平滚动浏览，也可点击左右两侧的箭头按钮进行翻页。

2.9 窗口缩放

主界面窗口支持拖拽边缘进行等比缩放。将鼠标移动到窗口四个边缘或四个角，当光标变为双向箭头时按住拖拽即可调整窗口大小。窗口内所有元素（文字、按钮、图标、卡片等）同步等比缩放，保持布局协调。

2.10 快捷键

Ctrl+F9 —— 开始/停止录制。切换录制的开始和停止状态。
Ctrl+F10 —— 暂停/继续录制。录制中切换暂停和继续状态。
Ctrl+Esc —— 退出程序。仅在未录制状态下有效。

以上快捷键为全局快捷键，无论当前焦点在哪个应用程序，均可触发相应操作。

四、视频输出

4.1 输出格式

视频格式：MP4（H.264 编码）。
音频格式：与视频合并封装在 MP4 中，或单独导出为音频文件。

4.2 输出路径

所有录制文件默认保存在当前用户的"视频"文件夹下的"录屏王"子目录中，即：

此电脑 ＞ 视频 ＞ 录屏王

文件命名规则如下：
- 全屏录制（单显示器）：recording_screen1_20260417_143025.mp4
- 全屏录制（多显示器）：recording_screen1_时间戳.mp4、recording_screen2_时间戳.mp4，编号依次递增
- 区域录制：recording_20260417_143025.mp4（无屏幕编号）
- 时间戳格式为：年月日_时分秒


五、常见问题

问：录制时画面卡顿怎么办？
答：录屏王会自动检测并优先使用硬件编码器（NVIDIA NVENC、AMD AMF、Intel QSV），如未检测到硬件编码器则使用软件编码。确保显卡驱动为最新版本可提升录制流畅度。

问：录制的视频在哪里？
答：所有视频保存在"视频 ＞ 录屏王"文件夹中。也可在主界面底部预览区点击文件夹图标直接打开。

问：免费版有什么限制？
答：免费版录制时长限制为90秒，录制的视频带有软件水印，区域录制功能不可用。开通会员后所有限制自动解除。

问：支持哪些操作系统？
答：目前仅支持 Windows 10 和 Windows 11 操作系统。

问：如何更换绑定的电脑？
答：会员授权与当前电脑硬件绑定。如需更换电脑，请通过客服渠道联系处理。
